CONFIGURACIÓN PARA GUARDAR SONIDOS DE AUDIO
Este tipo de pregunta permite al alumno grabar sonidos de audio mediante un plugin de java. El plugin de java es una pequeña aplicación de java que se ejecuta en la JVM (máquina virtual de java) del ordenador del usuario. Por defecto, la JVM tiene unas políticas de seguridad que impiden guardar archivos de sonido. Por ello es necesario que cambiemos esas políticas de seguridad para permitir guardar sonidos de audio.
Eso se puede hacer de dos maneras:

  1. Firmando, mediante un certificado digital, el archivo jar del plugin. Esta es la opción ideal, pero el problema es que, por el momento, la UPV no dispone de un certificado digital.
  2. Editar el fichero donde se guarda la política de seguridad de la JVM. Esta es la opción que vamos a utilizar. Es necesario hacer los siguientes pasos.

PASOS

En primer lugar, localizaremos el directorio donde se ha instalado la JVM que utiliza el sistema operativo. Habitualmente, en los sistemas Windows se suele encontrar en el directorio:
C:\Archivos de programa\Java\jre6\lib\security
En segundo lugar pulsaremos el botón derecho del ratón sobre el archivo "java.policy" y seleccionaremos "Edit with Notepad++" como mostramos en la figura 1.

Figura 1
Finalmente editaremos el archivo añadiendo al final de éste las siguientes líneas de código (Vea figura 2):
grant {
permission javax.sound.sampled.AudioPermission "record";
permission java.io.FilePermission "<<ALL FILES>>", "read, write";
permission java.util.PropertyPermission "user.dir", "read";
};
